# Zoom Schedule

This Schedule helps students to keep all their Zoom-Links and passwords at one place and access them from every device.

# Setup - Build - Run

## Install all dependencies

```
yarn
```

## Run development server

```
yarn start
```

## Build project

```
yarn build
```
